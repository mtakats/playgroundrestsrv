# Playground Rest Service

Demo REST service with Spring Boot

## Useful commands

* mvnw clean package
* mvnw test
* mvnw verify
* mvnw spring-boot:run

## Endpoints

* http://localhost:8080/swagger-ui.html

## Static pages

* http://localhost:8080/index.html
* http://localhost:8080/jsDemo.html
